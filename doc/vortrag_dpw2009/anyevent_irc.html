AnyEvent::IRC - Event basierte IRC Toolbox für Perl

AnyEvent::IRC - Event basiert IRC Toolbox für Perl

Robin Redeker

Was ist IRC (Internet Relay Chat)?

  • Ein (altes) Realzeit Chat Protokoll für Internet.
  • Server verbunden in einer Sterntopologie.
  • Alt: RFC 1459 ist von 1993.
  • 2000 gabs dann die neuen IRCnet RFCs: 2810-2813.
  • Seit 1993 entwickelten IRC Server/Netze unabhängig erweiterungen.

AnyEvent::IRC

  • Ein (pure) Perl Modul für das IRC Client Protokoll.
  • 100% non-blocking (dank AnyEvent::Socket & AnyEvent::Handle).
  • Event basiert.
  • Aufspaltung:
    • AnyEvent::IRC::Client highlevel client API.
    • AnyEvent::IRC::Connection low-level Protokoll handler.

Neues von AnyEvent::IRC

  • Ursprünglich Net::IRC3.
  • Callback Registrierungs API Object::Event (u.a. benutzt von AnyEvent::XMPP).
  • ISUPPORT Unterstützung (Auto Erkennung des Case-Mappings und div. neuer Protokoll Erweiterungen).
  • 100% Non-blocking (Net::IRC3 hatte blocking DNS).
  • Diverse API Korrekturen.

Beispiel - Script

#!/opt/perl/bin/perl
use AnyEvent;
use AnyEvent::IRC::Client;

my $con = AnyEvent::IRC::Client->new;
$con->reg_cb (
   connect => sub {
      # Fehlerbehandlung etc...
   },
   privatemsg => sub {
      my ($con, $nick, $msg) = @_;
      print "Private Nachricht von $msg->{prefix}:\n"
           ."$msg->{params}->[1]\n";
   }
);
$con->connect ("irc.freenode.net", 6667, { nick => "elmex" });
AnyEvent->condvar->recv; # Event loop starten
